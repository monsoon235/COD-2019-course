# Lab5 多周期 MIPS CPU

## 逻辑设计

本实验设计的 MIPS 是多周期的，共支持以下种类的指令：

- 访存类：`lw`, `sw`

- R-R 运算类：`add`, `sub`, `slt`, `sltu`, `sll`, `srl`, `sra`, `sllv`, `srlv`, `srav`, `and`, `or`, `xor`, `nor`

- R-I 运算：`addi`, `andi`, `ori`, `xori`, `slti`, `sltiu`

- 跳转指令：`j`, `jal`, `jr`, `jalr`

- 分支指令：`beq`, `bne`, `blez`, `bgtz`, `bltz`, `bgez`
